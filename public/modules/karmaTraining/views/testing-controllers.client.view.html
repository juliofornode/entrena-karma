Testing Controllers

Basic parts of a test:

  1. describe('name', callback)
  2. it('should...', callback)
  3. expect(x).toBe()

Common steps before running a controller test:

  1. Load the module that contains the controller tested: beforeEach(module('moduleName'));
  2. Use inject to access _$controller_, the service that is responsible for instantiating controllers: beforeEach(inject(_$controller_)
  3. We assign _$controller_ to the new variable $controller.
  4. We create the scope object: var scope = {}
  5. We create an instance of the controller and assign value to its parameters: var ControllerName = $controller('ControllerName', {$scope: scope})
  6. We assign the scope model (data).
  7. Using assertions, we start testing scope variables and functions.
