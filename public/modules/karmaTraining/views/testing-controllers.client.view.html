Basic parts of a test:

  1. describe('testName', callback)
  2. it('should return...', callback)
  3. expect(scope.x).toBe('y')

Common steps before running a controller test:

  1. Load the module that contains the controller tested: beforeEach(module('moduleName'));
  2. Use inject to access _$controller_, the service that is responsible for instantiating controllers: beforeEach(inject(function(_$controller_) {...
  3. Inside the injected function, assign _$controller_ to the new variable $controller.
  4. On every "it", create the scope object: var scope = {}
  5. On every "it", create an instance of the controller and assign value to its parameters: var ControllerName = $controller('ControllerName', {$scope: scope})
  6. On every "it", assign the scope model (data).
  7. Using assertions, we start testing scope variables and functions.
