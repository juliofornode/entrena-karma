The purpose of this site is to train with Karma.js.


Client-side testing in Mean.js

  1. We play the configured tests running grunt test
  2. I created a custom grunt testclient to run just karma tests.
  3. Meanjs uses a separate db for testing. It is referred in config/env/test.js
  4. The tests generated for our custom crud module are located in public/modules/items/tests

Intro to Karma

  1. Framework to test AngularJS code. Created by the original AngularJS team.
  2. It runs on top of an assertion library. The most commonly used is Jasmine.
